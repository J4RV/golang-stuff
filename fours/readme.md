## Simple "Four in a row" terminal game

To play: 

```
go install github.com\j4rv\gostuff\fours  
fours
```

Example game:

```
[                 ]
[                 ]
[    □ ■          ]
[    □ □          ]
[■   ■ ■   ■      ]
[□ □ □ ■   ■ □    ]
 0 1 2 3 4 5 6 7 8
Current player: WHITE (□)
Enter a column index:
```
